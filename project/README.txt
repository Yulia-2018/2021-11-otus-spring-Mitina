### Проект 
#### Тема "Система управления личными задачами"

##### Основной функционал приложения:

Пользователи управляют своими личными задачами. Пользователи могут выполнять CRUD операции со своими задачами. Каждый пользователь имеет доступ только к своим задачам. http://localhost:8080.
В приложении есть телеграм бот, который каждый день в 9 часов утра рассылает пользователям их список задач на сегодня.

Новый пользователь может зарегистрироваться в системе. http://localhost:8080/register.

Данные о пользователях может просматривать только администратор. http://localhost:8080/rest/user.

В проекте используется база H2

Используемые технологии для написания backend:

- Spring Boot
- Spring MVC
- Spring Security
- Spring Data JPA
- Spring Validation
- Spring Boot Test
- Spring Security Test
- Lombok

Используемые технологии для написания frontend:
- Thymeleaf
- JQuery
